# Folder Size Calculator

A Python script that calculates the total size of a folder and all its files.

## Description
This program scans a given folder and sums the size of every file inside it, including files in subfolders.

The final size is displayed in megabytes (MB).

## Features
- Calculates folder size
- Includes subdirectories
- Simple command-line tool
- Beginner-friendly implementation

## Technologies Used
- Python
- os module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python folder_size.py

4. Enter the folder path when prompted

## Example
Input:
Total folder size: 24.35 MB
